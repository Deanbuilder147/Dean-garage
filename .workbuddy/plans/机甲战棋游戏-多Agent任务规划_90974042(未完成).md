3. **地图服务 (3002)**: 战场创建/编辑/删除、六角格坐标、基础地形编辑API
4. **战斗服务 (3003)**: 基础移动/攻击、回合管理、伤害计算、奇袭系统(后端逻辑)、阵营技能(后端逻辑)

6. **前端 (8081)**: Vue3+PixiJS、8个视图、CRT风格UI

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                      浏览器（用户）                      │
│   Vue3 + PixiJS（画战场、显示棋子）                       │
└─────────────────────┬───────────────────────────────────┘
                      │ 网络请求
┌─────────────────────▼───────────────────────────────────┐
│                    服务器                               │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐                │
│   │ 认证服务 │  │ 战斗服务 │  │ 地图服务 │                │
│   │  3005   │  │  3003   │  │  3002   │                │
│   └─────────┘  └─────────┘  └─────────┘                │
│   ┌─────────┐  ┌─────────┐                            │
│   │ 通讯服务 │  │格纳库服务│                            │
│   │  3004   │  │  3001   │                            │
│   └─────────┘  └─────────┘                            │
│   SQLite 内存数据库                                     │
└─────────────────────────────────────────────────────────┘
```

## 技术栈

- **后端**: Node.js + Express + sql.js(SQLite内存数据库)
- **前端**: Vue3 + PixiJS + Pinia
- **通信**: RESTful API + WebSocket(Socket.io)
- **部署**: Docker + Docker Compose

## Skill分配

- **frontend-agent**: Impeccable（前端设计工具集）- 用于UI优化
- **其他服务**: 通用Node.js开发，无需特定skill

## Agent Extensions

### Skill

- **Impeccable（前端设计工具集）**
- Purpose: 优化前端UI界面，提供高品质设计能力，包括视觉设计、布局排版、动效交互等
- Expected outcome: 提升游戏界面视觉质量，避免"AI审美"，实现独特、生产级的前端设计

### SubAgent

- **code-explorer**
- Purpose: 用于复杂代码库探索，搜索多个文件和目录
- Expected outcome: 快速定位相关代码文件，理解代码结构
