[SECTION: DESIGN_RULES]

设计规范

核心原则

规则 说明 实现
NO-LINE 禁止1px solid边框，使用色调层级区分 使用 `::before` 伪元素渐变
0PX_RADIUS 所有组件必须直角 `border-radius: 0px`
SPACE_GROTESK 标题和主要文本使用Space Grotesk `font-family: var(--font-headline)`
STEPS_ANIM 使用 `steps(4)` 模拟低帧率 `transition: steps(4)`
CRT_SCAN 所有主表面叠加CRT scanlines `.scanline-overlay`
PIXEL_BORDER 3px粗边框 + 多重阴影 `.pixel-border-primary`
