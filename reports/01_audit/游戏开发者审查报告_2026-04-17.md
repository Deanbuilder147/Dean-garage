## 📋 具体代码改进建议

### 1. 战斗伤害计算公式
**当前实现：** 使用 `DamagePipe` 但公式不透明

**建议公开公式：**
```javascript
/**
 * 伤害计算公式（文档化）
 * 
 * 基础伤害 = 攻击方 (格斗/射击) - 防御方 (格斗/射击)
 * 最终伤害 = 基础伤害 × 地形修正 × 词条修正 + 随机浮动 (±2)
 * 
 * 暴击判定：掷骰≥9 时伤害×1.5
 * 闪避判定：机动差≥3 时 20% 概率闪避
 */
function calculateDamage(attacker, target, attackType) {
  const attackStat = attackType === 'melee' ? attacker.格斗 : attacker.射击;
  const defenseStat = attackType === 'melee' ? target.格斗 : target.射击;
  
  let baseDamage = attackStat - defenseStat;
  if (baseDamage <= 0) baseDamage = 1; // 至少 1 点伤害
  
  // 地形修正
  const terrain = getTerrainAt(target.q, target.r);
  const terrainMod = terrain.defenseBonus ? 1 - (terrain.defenseBonus * 0.1) : 1;
  
  // 词条修正
  const tagMod = applyTagModifiers(attacker, target);
  
  // 随机浮动
  const randomMod = (Math.random() * 4) - 2; // -2 ~ +2
  
  const finalDamage = Math.floor(baseDamage * terrainMod * tagMod + randomMod);
  return Math.max(1, finalDamage);
}
```

### 2. AI 行为树
**当前实现：** 简单 AI 在 `phase9-ai-combat-demo.cjs`

**建议完善 AI：**
```javascript
/**
 * AI 决策优先级
 * 1. 检查是否可以击杀敌方高价值目标
 * 2. 检查是否需要撤退（HP<30%）
 * 3. 检查是否可以占领基地
 * 4. 默认：攻击最近敌人
 */
class AIBehaviorTree {
  decide(unit, battleState) {
    // 1. 击杀检查
    const killTargets = this.findKillTargets(unit, battleState);
    if (killTargets.length > 0) {
      return { action: 'attack', target: this.selectHighValueTarget(killTargets) };
    }
    
    // 2. 生存检查
    if (unit.hp < unit.max_hp * 0.3) {
      return { action: 'retreat', target: this.findSafePosition(unit, battleState) };
    }
    
    // 3. 目标检查
    const objectives = this.findObjectives(unit, battleState);
    if (objectives.length > 0) {
      return { action: 'move', target: objectives[0] };
    }
    
    // 4. 默认攻击
    const nearestEnemy = this.findNearestEnemy(unit, battleState);
    return { action: 'attack', target: nearestEnemy };
  }
}
```
