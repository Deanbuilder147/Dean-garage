### 方案 2: 生成 cells 数组
需要从 terrain 数据生成 hex cells 数组:

```javascript
function generateCells(width, height, terrain) {
  const cells = [];
  for (let q = 0; q < width; q++) {
    for (let r = 0; r < height; r++) {
      const key = `${q},${r}`;
      const terrainId = terrain[key] || 'lunar';
      cells.push({
        q, r,
        terrain: terrainId,
        unit: null
      });
    }
  }
  return cells;
}
```
