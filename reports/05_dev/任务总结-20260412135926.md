## 五、游戏规则概要

### 5.1 棋子结构

```
┌─────────────────────────────────────────┐
│              完整棋子（Unit）            │
├──────────┬──────────────────────────────┤
│  主机体   │  必须（机体类型，40点）       │
│  - HP    │  结构 × 10                   │
│  - 技能  │  3个槽                        │
├──────────┼──────────────────────────────┤
│  跟随    │  可选（Royroy类型，25点）      │
│  - HP    │  结构 × 3                     │
│  - 部署  │  主机体周围1-3格              │
│  - 技能  │  2个槽                        │
├──────────┼──────────┬───────────────────┤
│  左手    │  右手     │  其它             │
│ （可选）  │ （可选）   │  背包 或 载具    │
│武器/防具  │武器/防具  │  （可选）        │
│ 各15点   │ 各15点    │  各15点          │
└──────────┴──────────┴───────────────────┘
```

### 5.2 伤害计算公式

```
近战伤害 = 攻击方格斗 + (攻击方机动 - 防御方机动) + 武器加成 - 防具抵消 - 防御方防御
远程伤害 = 攻击方射击 + (攻击方机动 - 防御方机动) + 武器加成 - 防具抵消 - 防御方防御
```

### 5.3 阵营特性

| 阵营 | 特性 | 效果 |
|------|------|------|
| **地球联合** | 火力覆盖 | 全场一次，5×5范围所有单位-5HP |
| **拜隆** | 增援 + 月面回复 | 友军可分担伤害；月面地形每回合回复4HP |
| **马克西翁** | 奇袭 + 隐匿 | 跳过回合抢占攻击；未攻击单位不可被选中 |
