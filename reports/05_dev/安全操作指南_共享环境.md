## 🛡️ 安全操作规则 (Safe Operation Rules)

### ❌ 绝对禁止的命令 (NEVER RUN)

```bash
# 这些命令会杀死 Hermes Agent 导致会话断开
pkill node
killall node
pkill -f "node"
kill -9 $(pgrep node)
rm -rf node_modules  # 可能影响 Agent 依赖
```

### ✅ 安全的操作方式 (SAFE ALTERNATIVES)

#### 1. 精准杀死特定服务 (Kill Specific Service Safely)

```bash
# ✅ 安全：只杀死特定端口的服务
lsof -ti:3004 | xargs kill -9 2>/dev/null

# ✅ 安全：只杀死特定服务目录的进程
pkill -f "combat-service.*node"

# ✅ 安全：使用服务名过滤
pgrep -af "combat-service" | grep -v hermes | awk '{print $1}' | xargs kill -9
```

#### 2. 重启单个服务 (Restart Single Service)

```bash
# ✅ 安全模式：先检查端口，再杀进程，再启动
PORT=3004
SERVICE_DIR=/home/agentuser/Dean-garage/services/combat-service

# Step 1: 找出并杀死占用端口的进程
lsof -ti:$PORT | xargs kill -9 2>/dev/null || true
sleep 2

# Step 2: 验证端口已释放
if ! lsof -ti:$PORT > /dev/null 2>&1; then
    echo "Port $PORT freed"
    # Step 3: 启动服务
    cd $SERVICE_DIR && node src/index.js > /tmp/combat.log 2>&1 &
    echo "Service restarted"
fi
```

#### 3. 验证进程身份 (Verify Process Identity)

```bash
# ✅ 在杀进程前始终验证
pgrep -af node | grep -v hermes | grep -v "Web3Hermes"

# 检查进程工作目录
ls -la /proc/<PID>/cwd
```

## 🔧 推荐的服务管理脚本 (Recommended Management Scripts)

### 创建服务管理脚本

```bash
# 文件：/home/agentuser/Dean-garage/scripts/manage-services.sh

#!/bin/bash

# 服务端口映射
declare -A SERVICES=(
    ["auth"]=3001
    ["hangar"]=3002
    ["map"]=3003
    ["combat"]=3004
    ["comm"]=3005
    ["battle"]=3006
    ["frontend"]=8081
)

# 安全杀死特定服务
kill_service() {
    local name=$1
    local port=${SERVICES[$name]}
    
    if [ -z "$port" ]; then
        echo "Unknown service: $name"
        return 1
    fi
    
    echo "Stopping $name service on port $port..."
    
    # 只杀死占用特定端口的进程
    PIDS=$(lsof -ti:$port 2>/dev/null)
    
    if [ -n "$PIDS" ]; then
        # 验证不是 Hermes Agent
        for pid in $PIDS; do
            CMDLINE=$(cat /proc/$pid/cmdline 2>/dev/null | tr '\0' ' ')
            if [[ "$CMDLINE" != *"hermes"* ]] && [[ "$CMDLINE" != *"Web3Hermes"* ]]; then
                kill -9 $pid 2>/dev/null
                echo "  Killed PID $pid"
            else
                echo "  ⚠️  Skipping PID $pid (Hermes Agent)"
            fi
        done
        sleep 2
        echo "Service $name stopped"
    else
        echo "No process on port $port"
    fi
}

# 启动特定服务
start_service() {
    local name=$1
    local service_dir="/home/agentuser/Dean-garage/services/${name}-service"
    
    if [ ! -d "$service_dir" ]; then
        echo "Service directory not found: $service_dir"
        return 1
    fi
    
    echo "Starting $name service..."
    cd "$service_dir" && node src/index.js > /tmp/${name}.log 2>&1 &
    echo "Service $name started (PID: $!)"
}

# 主菜单
case "$1" in
    stop)
        kill_service "$2"
        ;;
    start)
        start_service "$2"
        ;;
    restart)
        kill_service "$2"
        sleep 2
        start_service "$2"
        ;;
    status)
        echo "=== Service Status ==="
        for name in "${!SERVICES[@]}"; do
            port=${SERVICES[$name]}
            pid=$(lsof -ti:$port 2>/dev/null)
            if [ -n "$pid" ]; then
                echo "✅ $name (port $port): PID $pid"
            else
                echo "❌ $name (port $port): NOT RUNNING"
            fi
        done
        ;;
    *)
        echo "Usage: $0 {start|stop|restart|status} [service-name]"
        echo "Services: auth, hangar, map, combat, comm, battle, frontend"
        ;;
esac
```

## 📋 操作检查清单 (Operation Checklist)

### 在重启服务前 (Before Restarting Services)

- [ ] 确认 Hermes Agent 进程 PID (`pgrep -af hermes`)
- [ ] 确认目标服务端口 (`lsof -i :PORT`)
- [ ] 验证目标进程不是 Agent (`cat /proc/PID/cmdline`)
- [ ] 使用端口方式杀进程 (`lsof -ti:PORT | xargs kill`)
- [ ] 等待 2 秒确认端口释放
- [ ] 启动新服务实例
- [ ] 验证健康检查端点 (`curl localhost:PORT/health`)

### 在清理日志文件前 (Before Cleaning Logs)

- [ ] 不删除 `/tmp/*.log` 如果服务正在运行
- [ ] 使用 `truncate` 而非 `rm` 清空日志
- [ ] 保留最近 7 天的日志用于调试

## 🚨 紧急恢复流程 (Emergency Recovery)

### 如果意外杀死了 Hermes Agent

1. **会话将立即断开**
2. **重新连接 OrcaTerm**
3. **检查 Agent 是否自动重启:**
   ```bash
   pgrep -af hermes
   ```
4. **如果未运行，手动启动:**
   ```bash
   cd /home/agentuser/.hermes/hermes-agent
   source venv/bin/activate
   hermes
   ```

### 如果游戏服务全部崩溃

```bash
# 一键重启所有服务（安全方式）
cd /home/agentuser/Dean-garage
./scripts/manage-services.sh restart auth
./scripts/manage-services.sh restart hangar
./scripts/manage-services.sh restart map
./scripts/manage-services.sh restart combat
./scripts/manage-services.sh restart comm
./scripts/manage-services.sh restart battle
```
