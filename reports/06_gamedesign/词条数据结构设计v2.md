## 一、指令对照表

| 英文指令 | 中文含义 | 备注 |
|----------|----------|------|
| `id` | 词条唯一标识 | 英文小写，下划线连接 |
| `name` | 词条中文名称 | 显示用 |
| `trigger.phase` | 触发阶段 | 时机大类 |
| `trigger.timing` | 触发时机 | 具体时间点 |
| `conditions.required` | 触发前置条件 | 数组，需全部满足 |
| `conditions.required[].check` | 检查项 | 字段名 |
| `conditions.required[].value` | 固定值 | 直接比较 |
| `conditions.required[].ref` | 引用值 | 引用其他字段 |
| `conditions.required[].operator` | 操作符 | `==` / `!=` / `>` / `<` / `>=` / `<=` |
| `effects[].type` | 效果类型 | 效果分类 |
| `effects[].dice.required` | 是否需要掷骰 | `true` / `false` |
| `effects[].dice.sides` | 骰子面数 | 默认6 |
| `effects[].dice.condition` | 掷骰条件 | 成功条件 |
| `effects[].dice.outcomes` | 掷骰结果映射 | 点数→结果 |
| `effects[].dice.choices` | 掷骰选项列表 | 多个分支 |
| `effects[].dice.choices[].range` | 点数范围 | `[1, 4]` |
| `effects[].dice.choices[].result` | 结果标识 | 成功/失败 |
| `effects[].dice.choices[].effect` | 效果定义 | 详细效果 |
| `params.priority` | 执行优先级 | 数字越大越先执行 |
| `params.optional` | 是否可选触发 | `true`=玩家选择，`false`=强制 |
| `params.consumable` | 是否消耗性 | 触发后消失 |
| `params.interrupt` | 是否中断流程 | 中断后续处理 |
| `params.trigger_limit` | 触发次数限制 | 每轮/每场等 |
| `params.setup_phase` | 需提前声明的阶段 | 回合开始时等 |
| `params.dm_controlled` | 是否DM控制 | 空投等 |
| `attack_type` | 攻击类型 | `melee`=近战，`ranged`=远程 |
| `kinetic` | 实体伤害 | 物理攻击 |
| `energy` | 光束伤害 | 能量攻击 |
| `damage_type` | 伤害属性 | 实体/光束 |

### 1. 斩杀 (execute)

```javascript
{
  id: 'execute',
  name: '斩杀',
  
  // ========== 检索索引 ==========
  trigger: {
    phase: 'post_damage',           // 触发阶段：伤害结算后
    timing: 'after_melee'          // 触发时机：近战伤害结算后
  },
  
  // ========== 触发条件链 ==========
  conditions: {
    required: [
      { check: 'attack_type', value: 'melee', operator: '==' },
      { check: 'target_hp', value: 5, operator: '<' }
    ]
  },
  
  // ========== 效果定义 ==========
  effects: [{
    type: 'instant_kill',          // 效果类型：立即斩杀
    dice: {
      required: true,
      sides: 6,
      condition: '>= target_hp',
      outcomes: {
        success: 'target_dead',
        fail: 'continue'
      }
    }
  }],
  
  // ========== 执行参数 ==========
  params: {
    priority: 60,                  // 优先级
    optional: false,               // 强制触发
    consumable: false,             // 可重复触发
    interrupt: false
  }
}
```

**触发流程**：
1. 近战攻击伤害结算
2. 检查目标HP是否<5
3. 是 → 触发斩杀判定
4. 掷骰，点数>=目标剩余血量 → 斩杀成功

### 2. 决斗 (duel)

```javascript
{
  id: 'duel',
  name: '决斗',
  
  trigger: {
    phase: 'pre_attack',           // 触发阶段：攻击前
    timing: 'before_damage_calc'    // 伤害计算前
  },
  
  conditions: {
    required: [
      { check: 'target_in_range', value: true, operator: '==' },
      { check: 'self_in_target_range', value: true, operator: '==' },
      { check: 'self_hp', ref: 'target_attack_max', operator: '<' },
      { check: 'target_hp', ref: 'self_attack_max', operator: '<' }
    ]
  },
  
  effects: [{
    type: 'duel_resolution',       // 效果类型：决斗判定
    dice: {
      required: true,
      sides: 6,
      participants: ['attacker', 'defender'],
      outcomes: {
        'higher_wins': 'winner_continues_attack',
        'tie': 'both_dead'
      }
    }
  }],
  
  params: {
    priority: 10,
    optional: false,
    interrupt: true
  }
}
```

**触发流程**：
1. 攻击开始前
2. 检查双方是否都在对方射程内
3. 检查双方HP是否都低于对方最高攻击值
4. 是 → 进入决斗模式
5. 双方掷骰，点数大者攻击，平局同归于尽

### 3. 抢夺 (plunder)

```javascript
{
  id: 'plunder',
  name: '抢夺',
  
  trigger: {
    phase: 'post_damage',
    timing: 'after_damage_resolved'
  },
  
  conditions: {
    required: [
      { check: 'damage_dealt', ref: 'target_weapon_attack', operator: '>' },
      { check: 'target_has_weapon', value: true, operator: '==' }
    ]
  },
  
  effects: [{
    type: 'plunder_attempt',       // 效果类型：抢夺判定
    dice: {
      required: true,
      sides: 6,
      choices: [
        { range: [1, 3], result: 'fail', effect: null },
        { range: [4, 6], result: 'success', effect: { damage_modifier: -10, loot_weapon: true } }
      ]
    }
  }],
  
  params: {
    priority: 50,
    optional: true,
    choices: ['attempt', 'skip']
  }
}
```

**触发流程**：
1. 伤害结算后
2. 检查本次伤害是否>被攻击者武器攻击值
3. 是 → 触发抢夺判定（攻击者可选择）
4. 掷骰：
   - 1-3点：抢夺失败
   - 4-6点：伤害-10，获得该武器

### 6. 再动 (reattack)

```javascript
{
  id: 'reattack',
  name: '再动',
  
  trigger: {
    phase: 'on_kill',
    timing: 'immediate_after'
  },
  
  conditions: {
    required: [
      { check: 'target_faction', ref: 'enemy', operator: '==' },
      { check: 'extra_turn_used_this_round', value: false, operator: '==' }
    ]
  },
  
  effects: [{
    type: 'grant_extra_turn',      // 效果类型：给予额外回合
    scope: 'full_turn'
  }],
  
  params: {
    priority: 90,
    optional: false,
    trigger_limit: {
      type: 'per_round',
      max: 1
    },
    cooldown: {
      self_trigger: false
    }
  }
}
```

**触发流程**：
1. 击杀敌方单位时
2. 检查本轮是否已触发过再动
3. 否 → 立即获得额外回合
4. 额外回合结束后标记"已触发"

### 7. 援助 (assist)

```javascript
{
  id: 'assist',
  name: '援助',
  
  trigger: {
    phase: 'on_ally_attacked',
    timing: 'during_attack'
  },
  
  conditions: {
    required: [
      { check: 'ally_in_line_of_sight', value: true },
      { check: 'ally_faction', ref: 'self_faction', operator: '==' },
      { check: 'ally_is_being_attacked', value: true }
    ]
  },
  
  effects: [{
    type: 'assist_choice',          // 效果类型：援助选择
    choices: [
      {
        id: 'move_intercept',
        cost: 'give_up_move_action',
        effects: [
          { type: 'teleport_to_position', position: 'between_attacker_and_ally' },
          { type: 'share_damage', value: 5 }
        ]
      },
      {
        id: 'counter_attack',
        cost: 'give_up_combat_action',
        effects: [
          { type: 'counter_damage', target: 'attacker', value: 5 }
        ]
      }
    ]
  }],
  
  params: {
    priority: 70,
    optional: true,
    timing_lock: 'next_turn'
  }
}
```

**触发流程**：
1. 友军在视距内被攻击
2. 玩家可选择放弃下一个行动：
   - 放弃移动：移动到攻击方与受击方之间，分担5点伤害
   - 放弃战斗：对敌方造成5点反击伤害

### 8. 空投 (airdrop)

```javascript
{
  id: 'airdrop',
  name: '空投',
  
  trigger: {
    phase: 'round_start',
    timing: 'turn_2_and_after',
    activation: 'dm_roll'
  },
  
  conditions: {
    required: [
      { check: 'current_round', value: 2, operator: '>=' }
    ]
  },
  
  effects: [
    {
      type: 'spawn_items',
      dice: {
        required: true,
        sides: 6,
        item_count: 'dice_result'
      },
      item_types: ['weapon', 'armor'],
      position: 'random'
    },
    {
      type: 'trigger_if_occupied',
      condition: 'target_position_has_unit',
      trigger: 'luck'
    }
  ],
  
  params: {
    priority: 5,
    optional: false,
    dm_controlled: true
  }
}
```

**触发流程**：
1. 第二轮开始
2. DM掷骰，点数=生成武器/防具数量
3. 随机生成坐标放置
4. 若坐标有单位，回合开始时触发幸运

### 9. 联防 (formation_defense)

```javascript
{
  id: 'formation_defense',
  name: '联防',
  
  trigger: {
    phase: 'movement_check',
    timing: 'before_move_resolved'
  },
  
  conditions: {
    required: [
      { check: 'moving_unit_faction', ref: 'defending_units_faction', operator: '!=' },
      { check: 'blocking_units_count', value: 3, operator: '>=' },
      { check: 'blocking_units_aligned', value: 'horizontal', operator: '==' }
    ],
    formation: {
      type: 'line',
      orientation: 'horizontal',
      min_units: 3,
      spacing: 'adjacent'
    }
  },
  
  effects: [{
    type: 'block_movement',
    blocked: {
      directions: 'straight_line_through',
      exceptions: null
    },
    resolution: 'must_route_around'
  }],
  
  params: {
    priority: 95,
    optional: false,
    passive: true
  }
}
```

**触发流程**：
1. 移动判定时
2. 检查是否有3个同一阵营单位横向相邻排列
3. 是 → 非同阵营单位不能直线穿越，需绕行
