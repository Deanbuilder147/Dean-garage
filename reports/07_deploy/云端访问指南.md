## 🔧 故障排查

### 问题 1：无法访问网站
**症状：** 浏览器显示 "无法连接" 或 "连接超时"

**解决：**
1. 检查云服务器安全组是否开放 8081 端口
2. 尝试 ping 服务器：`ping 106.54.197.69`
3. 检查本地防火墙是否阻止

### 问题 2：页面加载但 API 请求失败
**症状：** 页面能打开，但点击按钮后显示网络错误

**解决：**
1. 检查浏览器控制台 (F12 → Console) 的错误
2. 确认所有后端服务运行正常
3. 查看服务日志：
   ```bash
   # 在云端执行
   tail -f /tmp/frontend.log
   tail -f /tmp/comm-service.log
   ```

### 问题 3：仍然显示 token 错误
**解决：**
1. 清除浏览器缓存 (F12 → Application → Clear site data)
2. 重新登录
3. 检查 Network 标签的请求详情
4. 如果仍不行，提供详细错误信息
